### Requirements

- Python3.5
- Virtual Env (venv)


### Exe

1 - Clone the repo


2 - Create the environment:
```
python3 -m venv enn

```

3 - Source into the environment:
```
source env/bin/activate

```

4 - Install Dependencies:
```
pip install --upgrade pip
pip install -r requirements.txt

```

5 - Execute:

```python
python main.py

```



### Reference
